# Auditoria de Dados - Toolkit

Este projeto fornece uma ferramenta simples em Python para auditar a qualidade de arquivos de dados (CSV e JSON).

## Funcionalidades
* Análise de valores ausentes (missing values).
* Detecção de outliers em colunas numéricas.
* Geração de relatório de diagnóstico em formato JSON.

## Como usar
No arquivo `main.py`, aponte para o seu arquivo de dados e execute o script para gerar o diagnóstico.
